# Sorting-Algorithm-Simulator
A Java Program to demonstrate Sorting Algorithms in GUI using Swing library.
Each algorithm here is implemented on threads and simulated using pause 'wait()' and play 'notify()' 

# Compile
javac *.java

# Execute/Run
java Home

# Contents
  1) Bubble Sort
  2) Selection Sort
  3) Insertion Sort
  4) Quick Sort
